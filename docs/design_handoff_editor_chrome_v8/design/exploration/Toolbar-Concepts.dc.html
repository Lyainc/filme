2툴바 모듈 · 1e 통합 최종안
1e를 기본으로 통합했습니다. 배치 변경은 툴바 안 ⚙ 설정에서만 하도록 햄버거를 없앴고, 설정 메뉴는 아래서 올라오지 않고 툴바에 붙어서 열립니다. 기본값은 좌측 세로 고정 레일이고, 목록은 툴바 위치와 무관하게 항상 우측 drawer로 열려요. 툴바 아이콘에 숨기기를 추가해 감추면 작은 원형 복원 버튼만 남습니다.
2a좌측 고정 레일 · 통합 설정 · 숨기기

1툴바 모듈 시안 · 방향 × 배치
에디터 상단 미니 툴바(실행취소 · 다시실행 · 항목 · 최대화)를 모듈만 분리해 방향(가로/세로)과 배치(고정/이동)로 탐색한 5가지 시안입니다. 각 폰에서 직접 손잡이를 드래그하고, 항목 아이콘으로 목록을 열어볼 수 있어요.
1a가로형 · 이동식
손잡이(⋮)를 잡고 포스터 위 어디로든 이동. 항목 아이콘 → 목록.
1b가로형 · 고정식
헤더에서 내려와 포스터 바로 위에 고정. 위치 고정형.
1c세로형 · 고정식
좌측 세로 레일 고정. 목록이 좌측에서 펼쳐집니다(본편은 우측).
1d세로형 · 이동식
세로형 툴바를 손잡이로 자유롭게 이동.
1e가로·세로 전환 + 유저 선택
⚙ 설정으로 1~4 배치를 직접 선택, 햄버거(≡)로 가로/세로 토글. (1~4 통합안)
